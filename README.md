# Todoist Clone

Pixel-perfect Todoist clone - Full-featured task management SPA

## Stack

React, Next.js, TypeScript, Tailwind CSS.

## What it includes

- Add Project Modal
- Add Task
- App Provider
- App Shell
- Quick Add
- Section Header
- Sidebar
- Task Detail
- Task Item
- Task List
- Use Store

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Run the local development server: `NODE_ENV=development next dev -p 8080` |
| `npm run build` | Create a production build: `next build` |
| `npm run start` | Start the production/server process: `next start -p 8080` |

## Local setup

```bash
npm install
npm run dev
```
